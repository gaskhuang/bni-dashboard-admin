# 華One Palms BNI 幹部管理儀表板

BNI 華One Palms 分會領導團隊專用的數據管理介面，透過數據全面掌握團隊狀況，找出需要關心或鼓勵的夥伴。

## 線上版本

https://gaskhuang.github.io/bni-dashboard-admin/

## 五大功能

### 1. 團隊概覽
- 團隊平均分、綠燈率、警戒人數、月平均分變化、全勤率
- 燈號分布橫條（綠/黃/紅/黑人數一目了然）

### 2. 會員績效
- 可排序的完整會員表格（總分、各項指標、趨勢）
- 篩選功能：依燈號篩選、搜尋姓名
- 迷你進度條顯示各項得分
- 點擊會員開啟詳細 Modal

### 3. 需關注會員
- **績效下滑警示**：最近 2 個月總分下降 ≥ 10 分
- **持續低分會員**：連續 3 個月以上低於 50 分
- **進步會員表揚**：最近 2 個月總分提升 ≥ 10 分

### 4. 數據分析
- **團隊平均分趨勢**：6 個月折線圖，含綠燈 70 分參考線
- **燈號總分統計**：每月各燈號人數堆疊長條圖
- **各項目得分率**：6 個 PALMS 類別的團隊平均得分率（得分率低 = 分會弱點）
- **分數分布**：直方圖依紅綠燈門檻著色

### 5. 例會準備
- 自動生成表揚名單、關懷名單
- 本月重點（團隊最弱項目 + 建議行動）
- 團隊目標（黃燈會員提升建議）

## 技術架構

- 純 HTML / CSS / JavaScript，無框架
- [Chart.js](https://www.chartjs.org/) 繪製圖表
- 資料來源為 Google Sheets 發佈的 CSV，與會員查詢系統共用同一資料源
- 託管於 GitHub Pages
